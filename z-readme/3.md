1️⃣ 同步 JS
2️⃣ 微任务（Promise / queueMicrotask / Vue nextTick）
3️⃣ requestAnimationFrame 回调
4️⃣ DOM 样式计算 + Layout
5️⃣ Paint（真正渲染到屏幕）✅ 第一次渲染完成
6️⃣ setTimeout / setInterval 回调（宏任务）

🔥 nextTick 是“框架层的 DOM 更新完成”
🔥 requestAnimationFrame 是“浏览器层的渲染时机确认”

【第 2 帧】
mousedown
├─ blur / focus（浏览器）
mousedown 回调执行 -> 进入回调内部：
├─ 同步 JS（DOM 更新）
├─ 微任务（nextTick）
blur 回调执行
├─ rAF
└─ 渲染 👉 第一次渲染

【第 3 帧】
mouseup
click
setTimeout
└─ DOM 更新
└─ 渲染 👉 第二次渲染
